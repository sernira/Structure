صفحه ی اصلی   Structure

فصل 1: ساختار بنیادین سند HTML

1.1 استانداردهای HTML و مسیر تکامل

  • WHATWG (Living Standard) در برابر W3C
  • مقایسهٔ HTML4، XHTML و HTML5

1.2 تعریف نوع سند و تأثیر آن در مرورگر

  • <!DOCTYPE html> و حالت‌های rendering
  • دسته‌بندی تگ‌ها: Void, Raw Text, Escapable Raw Text

1.3 تگ‌های اصلی ریشهٔ سند

  • <html lang="" dir=""> و نقش ویژگی‌های زبانی
  • ساختار اولیه با <head> و <body>

1.4 اصول نگارش معتبر HTML

  • تو‌در‌تویی (nesting) صحیح، حساسیت به حروف
  • درج توضیح‌ها: <!-- … -->

فصل 2: متادیتا و اتصال منابع خارجی

2.1 متادیتاهای ضروری صفحه

  • <meta charset="UTF-8">
  • <meta name="viewport" …>

2.2 اطلاعات توصیفی برای SEO و موتور جستجو

  • <title>
  • متا: <description>, <keywords>, <author>, <robots>

2.3 اطلاعات اجتماعی برای اشتراک‌گذاری در شبکه‌ها

  • <meta property="og:*"> برای Open Graph
  • <meta name="twitter:*"> برای Twitter Cards

2.4 بارگذاری منابع خارجی و بهینه‌سازی

  • <link rel="stylesheet">, <link rel="icon">
  • preload، prefetch، modulepreload

فصل 3: ساختار معنایی سند HTML

3.1 چارچوب اصلی صفحه وب

  • <header>, <nav>, <main>, <footer>, <address>

3.2 بخش‌بندی محتوای داخلی

  • <section>, <article>, <aside>, <div>

3.3 ساختاردهی تیترها و سرفصل‌ها

  • <h1> تا <h6> و سلسله‌مراتب معنایی

3.4 ناوبری درون‌صفحه‌ای با شناسه‌ها

  • href="#id" و تعریف id در عناصر هدف

فصل 4: عناصر متنی بلاکی

4.1 پاراگراف‌ها و عناصر جداکننده

  • <p>, <hr>

4.2 نمایش نقل‌قول‌ها

  • <blockquote cite="">, <q>

4.3 نمایش متون فرم‌دار و کدنویسی

  • <pre>, <code>, <samp>, <kbd>, <var>

4.4 فهرست‌های ترتیبی و توضیحی

  • <ul>, <ol>, <li> با ویژگی‌های start, reversed, type
  • <dl>, <dt>, <dd>

فصل 5: عناصر متنی خطی و معنابخش

5.1 عناصر پایهٔ درون‌خطی متن

  • <span>, <br>

5.2 تأکید معنایی و جلوهٔ بصری متن

  • <strong>, <em>, <mark>
  • <b>, <i>, <u>, <small>, <del>, <ins>

5.3 ارجاع‌دهی، اختصارات، تاریخ و داده

  • <a>, <cite>, <abbr>, <data>, <time>

5.4 نمایش متون چندزبانه و تغییر جهت نوشتار

  • <bdi>, <bdo dir="">

فصل 6: تصاویر و رسانه های چندرسانه ای

6.1 نمایش و بهینه‌سازی تصاویر

  • <img> با ویژگی‌های srcset, sizes, loading, decoding
  • <picture>, <source>

6.2 صوت و ویدئو در صفحات وب

  • <video>, <audio> و <track>
  • <source> برای چند فرمت

6.3 گرافیک برداری و گرافیک پیکسلی

  • <svg>
  • <canvas>

فصل 7: جاسازی محتوای خارجی و اشیاء

7.1 نمایش صفحات خارجی با iframe

  • <iframe> با تنظیمات امنیتی و دسترسی

7.2 فایل‌های خارجی با embed و object

  • <embed>, <object>, <param>

7.3 بررسی تگ‌های منسوخ‌شده HTML

  • <applet>, <frame>, <frameset>

فصل 8: نمایش داده‌ها با جدول

8.1 ساختار کامل جداول HTML

  • <table>, <caption>, <thead>, <tbody>, <tfoot>
  • <tr>, <th>, <td>, <colgroup>, <col>

8.2 ترکیب سلول‌ها در جدول

  • ویژگی‌های colspan, rowspan

8.3 جداول دسترس‌پذیر و معنایی‌تر

  • scope, headers, و summary (منسوخ‌شده)
  • جایگزینی با aria-describedby

8.4 تکنیک‌های واکنش‌گرایی در جداول

  • اسکرول افقی، ساختار کارت‌گونه، تغییر چینش

فصل 9: فرم‌ها و عناصر ورودی

9.1 ساختار کلی تگ فرم

  • ویژگی‌های مهم: action, method, enctype, autocomplete, novalidate

9.2 عناصر ورودی متداول

  • <input type="…"> با انواع پایه‌ای: text, email, url, tel, ...

9.3 کنترل‌های ورودی پیشرفته

  • <textarea>, <select>, <option>, <optgroup>, <datalist>, <button>, <output>, <meter>, <progress>

9.4 اعتبارسنجی فرم در HTML5

  • ویژگی‌هایی مانند required, pattern, min, max
  • معرفی اولیهٔ Constraint Validation API

9.5 بهبود دسترس‌پذیری فرم‌ها

  • <label> و ویژگی‌های ARIA مانند aria-label

فصل 10: ویژگی‌های عمومی و اسکریپت‌ها

10.1 ویژگی‌های جهانی برای تمام تگ‌ها

  • class, id, hidden, lang, tabindex, title, style, data-*, ...

10.2 استفاده از ویژگی‌های سفارشی (data-*)

  • تعریف و کاربرد در JavaScript

10.3 اسکریپت‌نویسی و رویدادها در HTML

  • <script> با async, defer, type="module"
  • هندلرهای درون‌خطی (inline event handlers)

فصل 11: دسترس‌پذیری ساختاری برای همه کاربران

11.1 چهار اصل اصلی WCAG

  • قابل‌ادراک (Perceivable)
  • قابل‌عمل (Operable)
  • قابل‌فهم (Understandable)
  • مقاوم (Robust)

11.2 نقش‌ها و صفات ARIA

  • roleهای ساختاری و ویژگی‌های aria-*

11.3 ناوبری بدون ماوس (کیبوردی)

  • tabindex, skip links و گردش در محتوا با کیبورد

فصل 12: داده‌های معنایی و ساخت‌یافته

12.1 نشانه‌گذاری Microdata با Schema.org

  • صفات itemscope, itemtype, itemprop

12.2 روش‌های RDFa و JSON-LD

  • مقایسهٔ نگارش و کاربرد

12.3 متاهای شبکه‌های اجتماعی پیشرفته

  • بازبینی کامل Open Graph و Twitter Cards

12.4 ویژگی‌های بهبود عملکرد بارگذاری

  • loading, decoding, crossorigin, referrerpolicy
